## Запуск проекта (Windows)

### Вариант 1. Через Git

```bat
git clone https://github.com/usrnmeee/LR-2-Semester
cd LR-2-Semester
scripts\setup_env.bat
```

### Вариант 2. ZIP + двойной клик

1. Скачайте ZIP с GitHub.
2. Распакуйте в любую папку.
3. Двойной клик на `scripts\setup_env.bat`.

# Извлечение данных (Extract)

Для загрузки данных используется скрипт `scripts/extract.py`, который читает параметры источника из конфигурационного файла и выполняет HTTP-запрос к API.

Пример запуска:

```bat
conda run -n data_env python scripts\extract.py configs\variant_09.yml
```

Скрипт:

* читает настройки из `/configs/variant_09.yml`;
* выполняет запрос к **World Bank API**;
* сохраняет исходный JSON-ответ в слой **raw**.

---

# Хранение Raw-данных

Raw-данные сохраняются в папку `/data/raw`.

Пример структуры:

```
data/
  raw/
    raw_YYYYMMDD_HHMMSS.json
```

Каждый запуск создаёт новый файл с временной меткой.
Сохраняется полный оригинальный ответ API для последующей обработки.

---

# Нормализация данных (Transform)

На этапе **Transform** raw JSON преобразуется в табличный формат.

Нормализация выполняется в ноутбуке:

```
notebooks/week3_eda.ipynb
```

Основные шаги:

1. загрузка raw JSON из `/data/raw`;
2. разворачивание структуры JSON в таблицу с помощью `pandas.json_normalize`;
3. переименование вложенных полей (`country`, `indicator`);
4. приведение типов данных (`date`, `value`);
5. удаление строк с отсутствующими значениями GDP;
6. удаление дубликатов.

---

# Хранение Normalized-данных

После обработки данные сохраняются в слой:

```
data/normalized/
```

Пример структуры:

```
data/
  normalized/
    normalized_20260305_102335.csv
```

Формат файла — **CSV**.

Каждый запуск нормализации создаёт новый файл с временной меткой.

---

# Структура Normalized-таблицы

Каждая строка таблицы соответствует:

**одному значению экономического индикатора для конкретной страны и года.**

Основные поля таблицы:

| Поле         | Тип    | Nullable | Источник в raw JSON | Описание                            |
| ------------ | ------ | -------- | ------------------- | ----------------------------------- |
| year         | int    | no       | `date`              | Год наблюдения                      |
| value        | float  | yes      | `value`             | Значение GDP в текущих долларах США |
| country_iso3 | string | no       | `countryiso3code`   | Трёхбуквенный ISO-код страны        |
| indicator    | string | no       | `indicator.id`      | Код экономического индикатора       |
